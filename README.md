# 教学管理系统

一个现代化的教学管理系统，用于管理教师、学生、课程安排和费用计算。

## 功能特点

### 核心功能

- **教师管理**：添加、编辑、删除教师信息，设置授课科目
- **学生管理**：维护学生信息，为每个学生设置每节课的固定费率
- **课程安排**：半小时精度的课程时间安排，支持一对一和小班课
- **费用计算**：自动按月统计费用，按学生费率和课程数量计算
- **数据备份**：支持数据导入导出，自动备份提醒功能

### 技术特性

- 纯前端应用，无需服务器
- 数据存储在浏览器本地
- 现代化Material Design风格界面
- 响应式设计，支持多种屏幕尺寸

## 技术栈

- **前端框架**: React 18 + TypeScript
- **UI组件**: shadcn/ui + Tailwind CSS
- **状态管理**: React Context API
- **数据存储**: LocalStorage
- **日期处理**: date-fns
- **构建工具**: Vite 5

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器


### 构建生产版本

## 使用说明

### 首次使用

1. 打开应用后，首先进入"教师管理"添加教师信息
2. 然后进入"学生管理"添加学生信息
3. 在"课程安排"中创建课程
4. 完成课程后，标记为"已完成"状态
5. 在"费用管理"中查看月度统计报表

### 数据管理

- **导出数据**：在仪表板或费用管理页面点击"备份数据"或"导出报表"
- **导入数据**：在仪表板点击"导入数据"，选择之前导出的JSON文件
- **数据位置**：数据存储在浏览器的LocalStorage中

### 课程类型说明

- **一对一**：一个教师对一个学生，费用按学生的每节课费率计算
- **小班课**：一个教师对多个学生，每个学生的费用按其自己的每节课费率除以班级人数计算

### 费率设置

- **学生费率**：每个学生设置固定的每节课费率（例如：学生A每节课100元，学生B每节课200元）
- **一对一课程**：学生支付其设定的每节课费率
- **小班课**：学生支付其每节课费率除以班级人数（例如：2人小班课，学生A支付50元，学生B支付100元）

## 数据备份建议

- 系统会在7天未备份时显示提醒
- 建议定期导出数据备份到本地文件
- 更换浏览器或清除缓存会导致数据丢失，请提前备份

## 项目结构

```
src/
├── components/          # UI组件
│   ├── common/         # 通用组件
│   └── ui/             # shadcn/ui组件
├── contexts/           # React Context
├── pages/              # 页面组件
├── services/           # 业务逻辑服务
├── types/              # TypeScript类型定义
├── utils/              # 工具函数
└── hooks/              # 自定义Hooks
```

## 注意事项

1. 数据存储在浏览器本地，请定期备份
2. 不要清除浏览器缓存，否则数据会丢失
3. 切换浏览器需要重新导入数据
4. 建议使用Chrome、Edge或Firefox等现代浏览器
