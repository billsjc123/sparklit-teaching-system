# 课程安排功能更新说明

## 新增功能

### 1. ✅ 数字输入框 - 学生费率
- 学生管理页面的"每节课费率"字段已经是数字输入框
- 支持小数点，可精确到分（0.01元）
- 最小值为0，防止输入负数

### 2. ✅ 学生选择 - 模糊搜索下拉框
- 使用 Combobox 组件（Command + Popover）
- 支持按学生姓名或年级搜索
- 多选功能，支持勾选/取消勾选
- 显示每个学生的费率信息
- 已选学生会显示在下方列表，可快速删除

**使用方法：**
1. 点击"搜索并选择学生"按钮
2. 在搜索框输入学生姓名或年级
3. 点击学生条目进行选择/取消选择
4. 选中的学生会显示✓标记

### 3. ✅ 科目选择 - 根据教师科目的下拉框
- 先选择教师后，科目下拉框会自动显示该教师的所有科目
- 切换教师时，科目选择会自动清空
- 如果教师没有科目，会提示"请先在教师管理中添加"

### 4. ✅ 课程复制功能 - 复制到后续N周
- 在列表视图和课程详情中都有"复制到后续周"按钮
- 可指定复制周数（1-52周）
- 自动检测时间冲突，跳过有冲突的时间段
- 复制完成后显示成功数量

**使用方法：**
1. 点击课程卡片或列表中的复制图标
2. 在弹出对话框中输入要复制的周数
3. 点击"确认复制"
4. 系统会自动在相同星期几、相同时间创建课程

### 5. ✅ 每天支持多节课程
- 月历视图中，每个日期格子可以显示多节课程
- 课程按开始时间排序显示
- 每节课显示为独立的卡片
- 有课程的日期底部显示"+ 添加课程"按钮，支持快速添加

### 6. ✅ 教师筛选功能
- 页面顶部新增教师筛选下拉框
- 可选择查看特定教师的课程，或显示所有教师
- 筛选后，日历视图和列表视图都会只显示该教师的课程
- 显示当前筛选条件下的课程数量

**使用方法：**
1. 在课程安排页面顶部选择教师
2. 日历和列表会自动更新
3. 点击日期添加课程时，会自动预选该教师

所有数据（教师、学生、课程）现在会：
- ✅ 自动保存到浏览器 localStorage
- ✅ 刷新页面后数据不会丢失
- ✅ 无需手动保存
- ✅ 支持数据导入导出备份

## 使用提示

### 创建课程的推荐流程
1. 选择教师（会自动加载该教师的科目）
2. 选择科目（从教师科目列表中选择）
3. 选择课程类型（一对一/小班课）
4. 搜索并选择学生（支持模糊搜索）
5. 设置开始和结束时间
6. 点击"创建"

### 批量安排课程
1. 创建第一节课程
2. 点击该课程的"复制"按钮
3. 输入要复制的周数（如 4 表示未来4周）
4. 系统会自动创建相同时间的课程

### 教师视图
1. 使用顶部的教师筛选器
2. 选择特定教师查看其课程表
3. 在该视图下添加课程会自动选中该教师
