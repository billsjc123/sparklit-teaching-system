# 月历视图功能说明

## 功能特点

### 1. 月视图日历
- 以月为单位显示课程安排
- 显示整个月的所有天数
- 每个日期格子显示当天的所有课程

### 2. 快速导航
- **上一月/下一月**：使用左右箭头按钮切换
- **本月**：快速跳转到当前月份
- **月份标题**：清晰显示当前查看的月份

### 3. 课程卡片展示
- **蓝色卡片**：待完成的课程
- **绿色卡片**：已完成的课程
- **卡片信息**：
  - 精确的上课时间（如：8:30 - 9:30）
  - 科目名称
  - 教师姓名（带👨‍🏫图标）
  - 学生姓名（带👨‍🎓图标）
  - 小班课标记
  - 完成状态标记（✓）

### 4. 灵活的时间支持
- **支持任意时间**：不限于整点，可以是 8:30、9:15 等任意时间
- **精确显示**：课程时间完整显示（如 8:30 - 10:00）
- **无需固定时间段**：只显示实际有课程的时间

### 5. 交互操作

#### 添加课程
1. **方式一**：点击空白日期格
   - 没有课程的日期显示"点击添加课程"
   - 点击后自动设置为当天9:00-10:00
   - 可在表单中自由修改为任意时间

2. **方式二**：点击日期右上角的"+"按钮（悬停显示）
   - 快速添加课程

3. **方式三**：点击右上角"添加课程"按钮
   - 手动选择日期和时间

#### 查看课程详情
- 点击课程卡片
- 弹出详情对话框，显示完整信息
- 可在详情中：
  - 标记完成/待完成状态
  - 删除课程

#### 切换视图
- **日历视图**：月视图日历，一目了然查看整月安排
- **列表视图**：传统表格形式，便于查看和管理所有课程

### 6. 视觉提示

#### 今日高亮
- 当天的日期数字显示为蓝色圆形背景
- 整个日期格子用浅蓝色背景高亮
- 方便快速定位当前日期

#### 课程数量统计
- 头部显示当月总课程数量
- 一眼看到本月工作量

#### 悬停效果
- 鼠标悬停在日期格子上会变色
- 提示可以点击添加课程

#### 课程类型标记
- 小班课显示"小班课"标签
- 一对一课程不显示特殊标签

## 使用场景

### 适合日历视图的场景
- 查看整月课程安排
- 了解每天的课程分布
- 寻找空闲日期
- 规划未来课程
- 直观查看工作量

### 适合列表视图的场景
- 查看所有课程记录
- 查找特定课程
- 批量查看课程状态
- 按时间排序查看

## 时间灵活性

### 支持的时间格式
- ✅ 整点时间：8:00 - 9:00
- ✅ 半小时：8:30 - 9:30
- ✅ 15分钟：8:15 - 9:15
- ✅ 任意分钟：8:23 - 9:47
- ✅ 跨小时：8:30 - 10:00

### 示例
```
8:00 - 9:00    ✓ 标准一小时课程
8:30 - 9:30    ✓ 半点开始
9:15 - 10:30   ✓ 15分钟开始，1小时15分钟课程
14:00 - 15:30  ✓ 下午课程
19:30 - 21:00  ✓ 晚上课程
```

## 技巧建议

1. **使用月视图规划课程**
   - 快速浏览整月安排
   - 找到空闲日期
   - 避免日程冲突

2. **多课程显示**
   - 一天内的多节课程会纵向排列
   - 按时间顺序自动排序
   - 每个课程都清晰可见

3. **课程状态管理**
   - 及时标记已完成的课程（变绿色）
   - 便于后续费用统计
   - 一眼区分完成状态

4. **小班课识别**
   - 小班课会显示所有学生姓名
   - 带有"小班课"标签
   - 便于区分课程类型

## 快捷方式

- 点击"本月"按钮：快速回到当前月份
- 点击空白日期：快速添加课程到该日期
- 点击课程卡片：查看详细信息并操作
- 使用左右箭头：快速浏览其他月份
